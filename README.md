# jeng-kata

Collection of programming problems.  Over time I've done quite a few of these on different platforms and in different contexts.  I wanted to start keeping them in one location.

I'm not stating that these solutions are optimal.  Surprisingly, I still have fun programming sometimes and do silly stuff just to see if it passes the test suite or to try something different.

Over time I plan to go back through and add the runtime analysis of the solutions.  I always have a lot of different personal projects that I bounce between.  Time will be the judge of how far I get with that.
